# Personal Dashboard — Спецификация проекта

Персональный дашборд для отслеживания обучения по нескольким трекам, управления задачами и расписанием, с AI-ассистентом на базе Claude.

---

## Стек технологий

- **Next.js** (App Router) + **TypeScript**
- **Tailwind CSS** + **shadcn/ui** — UI компоненты
- **Zustand** + **localStorage** — состояние и хранение данных
- **date-fns** — работа с датами
- **Claude API** через `/api/chat` route (серверный — ключ не в браузере)
- **Vercel** — деплой

---

## Страницы и маршруты

| Маршрут | Описание |
|---------|----------|
| `/` | Dashboard — главная страница с обзором дня |
| `/chat` | AI чат с Claude |
| `/schedule` | Расписание по дням недели |
| `/tasks` | Список задач |
| `/stats` | Статистика прогресса |

## Визуальный дизайн

- **Тема:** тёмная
- **Фон:** `#0a0a0f`
- **Карточки:** `#111118`
- **Шрифт:** Inter
- **Компоненты:** shadcn/ui с кастомными цветами

---

## Принципы UX

- Всё взаимодействие через чат (никаких форм — только разговор с AI)
- AI понимает контекст (задачи, расписание, треки) и помогает управлять ими
- Данные хранятся локально в браузере (localStorage)

---

## Порядок разработки

1. **Dashboard** — главная страница с обзором дня
2. **AI чат** — `/api/chat` route + UI интерфейс
3. **Расписание** — страница `/schedule`
4. **Задачи** — страница `/tasks`
5. **Статистика** — страница `/stats`

---

## Переменные окружения

Создать файл `.env.local` в корне проекта:

```
ANTHROPIC_API_KEY=your_api_key_here
```

## Запуск проекта

npm run dev
